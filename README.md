# Projects Server
## Mongo Half Stack app built as a student exercise for Alchemy Code Lab

## Author: Charly Welch
<blwbiology@gmail.com>

## Description
Save projects to a mongo database. Projects have the following properties:
- name
- budget
- timeline
- _id added by Mongo

## Dependencies:
- chai
- chai-http
- eslint
- mocha
- nodemon
- dotenv
- mongodb
